StoreSense is a retail analytics dashboard built using Power BI to analyze sales, profit, customers, payment modes, and product performance through interactive visualizations.

## Key Features
- Sales, Quantity, Profit & AOV KPIs
- State-wise sales analysis
- Monthly profit trend
- Payment mode distribution
- Category & Sub-category performance
- Interactive slicers and filters

## Design Approach
This version includes a creative visual background as a design experiment. A clean, professional interview-ready version is also under development.

- Business Analytics
